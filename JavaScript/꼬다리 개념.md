#### 코딩하다가 헷갈리는 개념 모음 🎈🎈

- Element
```js
//HTML이 다음과 갈을때!
<div id="content">
  <ul id="list">  
    <li>딸기</li>
    <li>망고</li>
</div>
```
- innerHTML :  div태그 안에 ul태그 부터 해당 
```text
  <ul id="list"> 
     <li>딸기</li>
      <li>망고</li>
  </ul>
```
- outerHTML : div태그부터 해당
```text
<div id="content">
      <ul id="list">  
        <li>딸기</li>
        <li>망고</li>
    </ul></div>
```
- textContent : 태그 안에 텍스트만 해당
        딸기
        망고
